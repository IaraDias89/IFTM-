Estrutura Básica de um Documento

  • !DOCTYPE html: Define que o documento é HTML5.
  • html: O elemento raiz que envolve todo o conteúdo.
  • head: Contém metadados, links para CSS e o título da aba.
  • body: Onde fica todo o conteúdo visível para o usuário.

Tags Mais Comuns

h1 a h6: Representam títulos de diferentes níveis de importância, sendo o h1 o título principal da página.

p: Utilizado para definir um parágrafo de texto comum.

strong ou b: Aplicam o efeito de negrito ao texto; a tag strong é usada quando há uma importância semântica no conteúdo.

em ou i: Aplicam o efeito de itálico ao texto; a tag em é usada para dar ênfase semântica a uma parte do conteúdo.

Listas, Links e Mídia

a: Utilizada para criar links, sendo obrigatório o uso do atributo href para definir o endereço de destino.

ul e li: Utilizadas para criar listas não ordenadas, que aparecem com marcadores de pontos ("bolinhas").

>

ol e li: Utilizadas para criar listas ordenadas, onde os itens aparecem numerados

img: Utilizada para inserir imagens na página, sendo obrigatório o uso do atributo src para indicar o caminho do arquivo de imagem.

Tags Semânticas (Boas Práticas)


  • header: Define o cabeçalho do site ou de uma seção específica, geralmente contendo logos e títulos.
  • nav: Indica o bloco que contém o menu de navegação e links principais.
  • main: Representa o conteúdo principal e único da página, onde fica o assunto central.
  • article: Utilizado para conteúdos que podem ser distribuídos de forma independente, como um post ou uma notícia.
  • section: Define uma seção temática dentro do documento para organizar diferentes partes do conteúdo.
  • footer: Define o rodapé da página, onde geralmente ficam informações de contato, direitos autorais e links secundários.
